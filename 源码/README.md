# TeX 源码说明

这里整理 GitHub Pages 中 PDF 对应的主要 TeX 源码和编译依赖。

- `主讲义/`：`main.tex`、AJbook 模板、样式文件、参考文献和章节源码。
- `第五六章重写/`：第五、六章重写稿入口文件。
- `习题答案/`：普通版和压缩版习题答案源码。
- `考试cheating paper版笔记/压缩版讲义/`：压缩版讲义入口文件及压缩章节源码。
- `考试cheating paper版笔记/四页手写总结/`：手写总结排版源码及对应图片素材。
- `Overleaf/`：可直接上传 Overleaf 的源码压缩包。

PDF 成品见仓库根目录下的 `解析数论基础/`。
